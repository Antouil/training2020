# web

这一章将基于 EESAST 目前网站方面使用的技术进行培训，内容主要为以下几个部分：

- Html, CSS, JavaScript/TypeScript 基础
- Node.js 与包管理
- React + antd
- Restful API 与 Express
- GraphQL + Hasura + Apollo
- 数据库 MongoDB 和 PostgreSQL

具体内容安排请参照各小节说明
